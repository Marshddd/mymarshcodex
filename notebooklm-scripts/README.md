# NotebookLM Scripts - แยกตามบท

ให้ใช้ทีละไฟล์กับ NotebookLM เพื่อให้ได้คลิปแยกครบ 6 บท

- lesson-01.md = บทที่ 1
- lesson-02.md = บทที่ 2
- lesson-03.md = บทที่ 3
- lesson-04.md = บทที่ 4
- lesson-05.md = บทที่ 5
- lesson-06.md = บทที่ 6

แต่ละไฟล์มี 3 ส่วนครบ:
1. คำพูดสอนสำหรับเสียง
2. ข้อความขึ้นจอสำหรับวิดีโอ
3. โค้ดตัวอย่างและคำอธิบาย

แนะนำให้สร้างเสียงและวิดีโอทีละบท แล้วนำลิงก์คลิปกลับไปใส่ในหน้า Admin ของเว็บตามบทเรียนนั้น ๆ
