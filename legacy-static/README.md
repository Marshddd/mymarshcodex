# Legacy Static Version

โฟลเดอร์นี้เก็บเว็บเวอร์ชันเก่าแบบ Static (`index.html`, `style.css`, `app.js`) ไว้เป็นต้นแบบ/สำรองเท่านั้น

โปรเจกต์หลักที่ใช้ deploy จริงคือ Next.js ที่ root ของโปรเจกต์:

- `app/`
- `components/`
- `lib/`
- `package.json`

ถ้าจะ deploy Render / ใช้ Admin / ใช้ MongoDB ให้ใช้ Next.js ตัวหลัก ไม่ใช่ไฟล์ในโฟลเดอร์นี้
